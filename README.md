# React Code & Express API Server

******************요약 및 의의******************

- 클론 코딩에서 배운 리액트 환경 구성 및 채팅 구현 응용을 위한 프로젝트
- 채팅과 상품 리스트 조회, 상세 조회 등 일부 페이지만 구현
- 클론 코딩 제공 Express 서버 분석을 통해 Express 서버 직접 구성
- Mongo DB 사용 방법을 학습하기 위해 채팅 메시지 Mongo DB로 구현
